# Halftone filter p5.js

Halftone effect made with p5.js (Processing).

# Setup

1. Install the server.

```bash
npm install -g http-server
```

2. Run the server.

```bash
http-server
```

3. In your browser open the page:
```
http://localhost:8080/
```
